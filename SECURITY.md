# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.x.x   | :white_check_mark: |
| < 4.0.0 | :x:                |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

If you discover a potential security vulnerability within Sift, please send an [e-mail](mailto:mirkoddd@gmail.com).

We will review your report, verify the vulnerability, and reply as soon as possible with a timeline for the fix. Once the fix is released, we will publicly acknowledge your contribution (if you wish).
